# Sign In - Sign up - React Native - Responsive
# Make By Dang Truong
## Tourial
1. Download or clone (`git clone https://github.com/dlmtruong1609/LoginReactNative.git`)
2. `npm install`
3. `npm run rename` change name project
4. `npm run android` to install app
5. `npm start`
